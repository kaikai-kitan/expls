このページでは、expls を実際に動かしながら使い方を覚えていきます。 むずかしい知識は必要ありません。順番に試してみてください。

ステップ1：そのまま実行してみる

オプションを何も付けずに実行すると、いまいるフォルダの中身が表示されます。

expls

ファイルは拡張子ごとに色分けされ、同じ種類のなかでは 更新が新しいものほど濃い色で表示されます。

ステップ2：隠しファイルも表示する（-a / –all）

. で始まるファイルやフォルダ（隠しファイル）は、通常は表示されません。 -a を付けるとそれらも表示できます。

expls -a

.git や .gitignore のような隠し項目も一覧に出てきます。

ステップ3：別のフォルダを見る（-p / –path）

いまいる場所を移動しなくても、-p で見たいフォルダを指定できます。

expls -p ./src

--path と書いても同じ意味です。指定しなければカレントディレクトリ（.）が対象です。

ステップ4：フォルダの中まで掘り下げる（-d / –depth）

-d に数字を渡すと、サブフォルダの中まで何段階潜るかを指定できます。 潜った中身はインデント（字下げ）されて表示されます。

expls -d 1   # 1段階だけ中に入る
expls -d 2   # 2段階まで入る

-d を付けなければ、サブフォルダの中までは表示しません（一覧だけ）。

ステップ5：グラデーションを逆にする（–reverse）

ふつうは「新しい＝濃い／古い＝薄い」ですが、--reverse を付けると逆になります。

expls --reverse   # 古いものを濃く、新しいものを薄く

「昔からあるファイルを目立たせたい」ときに便利です。

ステップ6：困ったらヘルプ（-h / –help）

オプションを忘れたときは、ヘルプを表示しましょう。

expls --help

使えるオプションの一覧と説明が出ます。

オプション早見表

オプション短縮形種類はたらき
--all-aフラグ隠しファイルも表示する
--path <パス>-p文字列表示するフォルダを指定する
--depth <数値>-d数値サブフォルダを何段階表示するか
--reverse（なし）フラググラデーションの濃淡を逆にする
--help-hフラグ使い方を表示する

組み合わせる

オプションは自由に組み合わせられます。たとえば次は 「src フォルダを、隠しファイル込みで、2段階潜って表示」します。

expls -a -d 2 -p ./src
