## expls について

**expls**（エクスプレス）は、標準の `ls` コマンドのように使える
ファイル一覧ツールです。名前は **ex**pand × **ls** に由来します。

ファイルを拡張子ごとに色分けし、さらに更新日時が新しいものほど濃く、
古いものほど薄くグラデーション表示します。これにより、
「どの種類のファイルか」「どれが最近更新されたか」がひと目で分かります。

- **作者**: kaido iwata
- **由来**: expand × ls
- **ライセンス**: MIT
- **言語**: Rust

詳しい使い方は[チュートリアル]({{< ref "03-tutorial" >}})を、
仕組みは[色分けのしくみ]({{< ref "04-algorithm" >}})をご覧ください。
